Voc 상담 대화 요약
연관 API

상담 대화문을 요약하여 상담사가 대화의 중요 내용을 알 수 있게 해주는 기술

생성 요약(Abstractive Summarization) 기술은 AI를 활용하여 뉴스 기사, 멀티턴 대화 등을 요약하는 문장을 생성하는 기술이며, KT가 연구하는 상담 대화 요약 생성은 이중 상담에 특화된 기술입니다.

상담 대화는 일반적으로 고객의 요구사항과 상담사의 응답의 형태로 대화가 진행 됩니다.

KT 100번 상담의 예를 들면, "요금제를 바꿔주세요"와 같은 고객 질의 사항에 대해 고객과 상담사의 대화가 이어지며 "XXX 요금제로 변경 진행 했습니다." 와 같은 상담사의 응대 결과로 진행이 됩니다.

상담 대화 요약 생성 기술은 고객의 질의사항과 상담사의 응대를 적절하게 요약하는 문장을 생성합니다. 상담사는 다음 상담이 진행 될 때 이 전에 요약된 상담 결과를 보고 빠르게 히스토리를 이해하여 응대 시간을 절약하고 더 효과적으로 응대할 수 있도록 합니다.

기존 상담 요약 기술은 대화문 내에서 중요하다고 판단되는 문장을 추출하는 패턴 기반의 추출 요약 방식을 사용합니다. 하지만 서비스별로 지속적인 패턴 추가 작업이 필요하고, 단순 문장을 추출하는 방식으로는 전체 대화 내용을 파악하기 힘들다는 단점이 있습니다.

아래는 실제 KT 100번 상담 대화 사례에 대한 핸드폰 고장 관련 상담의 패턴 추출 요약 방식과 생성 요약 방식 결과입니다.

01. 패턴 추출 요약 결과

고객 {"아니요 왔는데요. 핸드폰을 떨어트려서 이쪽도 나가고", "핸드폰 이상해서 가지고 왔는데 여러 문제가 있다고 하셔 가지고 확인하고 진행하려고 전화 드렸어요.", "그건 기억하는데 기간이 혜택을 받을 수 있는 기간인가 싶어서 오래 됐어요"}

상담사 {"단말기 보험센터로 연결하겠습니다."}

02 문장 생성 요약 결과

핸드폰 보험 기간 확인을 요청하여 단말기 보험 가입이 완료되어 있는 것으로 확인된다고 안내한 후 보험센터로 연결해주었다.

이처럼 생성 요약 결과는 대화문 흐름에 맞춰 함축적으로 요약되어 대화의 핵심 내용을 파악하기 좋은데 반해, 패턴 기반 추출 요약 결과는 핵심 내용을 한번에 파악하기가 상대적으로 쉽지 않아 여러 번 확인해야 되는 경우가 생기게 됩니다.

KT 상담 요약 기술은 Large AI 모델의 풍부한 어휘 및 문장 생성 능력을 기반으로 하여 적은 데이터만으로도 대화의 중요 내용을 함축적으로 요약하고 자연스러운 문장을 생성하며, 서비스 확장에 용이한 장점을 가지고 있습니다.
