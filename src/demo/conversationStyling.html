대화 스타일링(사투리/키즈 반말체)
연관 API

대화 스타일링(Dialogue Styling) 기술은 AI에 페르소나를 부여하여 동일한 답변도 상황에 따라 다양한 말투로 변환하는 기술입니다.

예를 들어 강아지가 태어났다는 고객 발화에 대해, 일반 고객에게는 "강아지는 항상 귀엽죠", 키즈 고객에게는 "강아지는 엄청 귀여워", 경상도 지역 고객에게는 "강생이는 억수로 귀엽재"와 같은 스타일링된 답변을 제공하도록 합니다.

일반적인 챗봇 서비스에서는 질의에서 바로 답변을 생성하지만, 스타일링의 경우 처음부터 답변을 생성하지 않고 특정 문장의 의미를 유지한 상태로 문장을 변환하는 기술이라는 점에서 차이가 있습니다. 이 때문에 대화 스타일링 기술은 시스템 답변의 의미를 유지하면서 다양한 사용자 타겟으로 안정적으로 답변을 확장할 수 있다는 장점을 가지고 있습니다. 단순한 단어나 서술어 어미 변환 수준의 스타일링은 간단한 규칙으로도 변환이 가능하지만, 문장의 의미를 유지하면서 구문 변환이 필요한 경우에는 딥러닝 모델을 필요로 합니다.

KT는 Encoder-Decoder 방식으로 사전 학습한 대용량 언어 모델을 활용하고 전이학습 방식(Transfer Learning)으로 스타일링을 학습합니다.

딥러닝으로 변환된 문장은 어법상 잘못된 문장일 수도 있고, 원래 문장의 의미와는 다르게 변환될 수 있기 때문에 그대로 사용하지 않고 비문이나 의미 변형을 검증하는 기술을 추가로 진행합니다.

KT의 스타일링 기술은 비문이나 의미 변형을 검증하기 위한 언어 모델을 학습하고, 이를 이용하여 정확도 높은 스타일링 서비스를 제공합니다.

01

Dynamic 스타일링

일부 단어나 문장의 서술어 변환에만 그치지 않고, 원 문장의 의미를
유지한 채 문장의 구조 변환까지 가능합니다.

02

실시간 스타일링

자사전 학습된 대용량 언어 모델 기반의 딥러닝 기술을 사용하지만,
실시간 변환이 가능한 수준으로 최적화 되어있습니다.
