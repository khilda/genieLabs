의도 분류 (Fine Tuning)
연관 API

대규모 언어 데이터 신경망에 기반하여 파악하고자 하는 의도 문장을 학습시키고 분류하는 기술

의도 분류(Intent Classification)는 텍스트에 담긴 목적이나 상황 등을 매핑 및 분류하는 기술입니다.
일반적으로 챗봇/보이스봇 등 자연어 UX 기반 서비스에서 사용자와 상호 작용하기 위한 핵심 기술 중 하나이며, 사용자의 자연어 입력 기반으로 사용자의 의도 또는 원하는 목적을 찾을 수 있도록 해줍니다.
KT 의도 분류는 대규모 신경망에 기반하여 확률적 의도 해석을 진행하기 때문에 발화 실수나 인식 오류에도 강건한 기술입니다.

KT 의도 분류 기술은 다음과 같은 특징을 가지고 있습니다.

01. 대규모 언어 데이터로 학습한 신경망 기반의 해석기

뉴스/위키 등 일반 어휘(19GB)와 상담어휘(10GB)를
추가 학습하여 챗봇/보이스봇의 어휘가 풍부하게 반영된 신경망을 가지고 있습니다.

02 언어 지식의 자기 지도 학습 + 영역 지식의 자기 지도 학습

자기 지도 학습으로 풍부해진 일반 영역 해석 지능 위에
소규모의 사업/서비스 맞춤형 의도 학습을 통해 의도 영역 커버리지 확장이 가능합니다.

03. 사업 환경에 맞는
다양한 모델 제공

14M, 110M에서 11B(예정) 파라미터 크기까지 성능과
하드웨어 요구 사항을 고려한 다양한 모델 제공이 가능합니다.

01

챗봇/보이스복 특화 사전 언어 모델

상담 어취 반영을 통한 챗봇/보이스봇에 특화된 사전 언어 모델

02

소규모 데이터로 강건한 의도 분류

대규모 자가지도 학습을 통한 지도 학습 성능 강화

03

운영 환경에 맞는 모델 제공

운영 환경에 맞게 성능과 하드웨어 요구사항을 고려한 다양한 모델 제공 기능 (성능 Trade-Off)
